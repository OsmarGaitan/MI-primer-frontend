#MI primer Fronent
<p>
 Pasos para crea mi primer archivo HTML
</p>
- Abre tu visual estudio code
- Luego crea el archivo HTML
  <P>
      - Ve a Archivo > Nuevo Archivo en el menú superior
	  - Alternativamente, puedes usar el atajo de teclado Ctrl + N (en Windows/Linux) o Cmd + N (en Mac).
	  - También puedes abrir una carpeta de proyecto arrastrándola a la ventana de Visual Studio Code para organizar todos tus archivos en un mismo lugar.

<p>
</p>
- Escribe el código HTML:
  <P>
      Si quieres iniciar con la estructura básica de HTML5, escribe un signo de exclamación ! y presiona la tecla Tab. Visual Studio Code generará automáticamente la plantilla básica.
	 
- Guarda el archivo
  <P>
      Presiona Ctrl + S (Windows/Linux) o Cmd + S (Mac) para guardar.
	  Nombra tu archivo y asegúrate de que la extensión sea .html (por ejemplo, index.html). Esto es crucial para que el navegador lo reconozca como una página web.
	  
- Visualiza tu archivo
  <P>
      -Con Live Server: Si tienes la extensión "Live Server" instalada, haz clic derecho sobre el archivo en el explorador de archivos y selecciona Open with Live Server. Esto abrirá el archivo en tu navegador predeterminado y refrescará automáticamente la página cada vez que guardes los cambios.
	  - Sin Live Server: Puedes hacer doble clic en el archivo HTML directamente desde la carpeta donde lo guardaste para que se abra en el navegador predeterminado. .


# Ejemplo.md
**Codigo Utilizado **

```
!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8" />
    <meta name="viewport" content="width=device-width, initial-scale=1.0" />
    <title>Mi primera página</title>
    <link rel="stylesheet" href="style.css">
</head>
<body>
    <h1>Hola Mundo</h1>
    <p>Esta es mi primera página web en.</p>
```
